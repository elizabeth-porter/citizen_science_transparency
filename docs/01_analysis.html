Load Cleaned Data

The following code uses the datasets created in the Initial Exploration that precedes the Analysis section. The datasets are saved to CSV as data/2025-12-23_csp-pub.csv (each row is a unique citizen science project-publication combination) and data/2025-09-24_pub.csv (each row is an individual publication).

# Unique citizen science project-publication combinations (N = 377)
df_csp = read_csv("data/2025-12-23_csp-pub.csv")

# Individual publications (N = 277)
df_pub = read_csv("data/2025-09-24_pub.csv")

                         count), # reorder columns by desc count

Visualizations

Code 
df_csp %>% 
  group_by(publication_year, transparency_bin) %>%
  summarise(count = n(), .groups = "drop") %>%
  mutate(status = ifelse(transparency_bin == 1, "Transparent", "Not Transparent")) %>%
  ggplot(aes(x = factor(publication_year),
             y = count,
             fill = status)) +
  geom_bar(stat = "identity") +
  labs(x = "Publication Year",
       y = "Number of Cases") +
  scale_fill_manual(values = c("Transparent" = "#ffc000",
                               "Not Transparent" = "#156082")) +
  theme_minimal() +
  theme(axis.text.x = element_text(angle = 60, hjust = 1),
        legend.position = "bottom",
        legend.title = element_blank())
